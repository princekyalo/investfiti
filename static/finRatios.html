📊 7 Key Financial Metrics Calculator

📈 P/E Ratio

Price-to-Earnings - Measures valuation by comparing price to earnings

P/E = Price / EPS

📊 PEG Ratio

Price/Earnings-to-Growth - Adjusts P/E for growth expectations [citation:2]

PEG = (Price / EPS) / Growth Rate

💰 PEGY Ratio

P/E to Growth & Yield - Includes dividends for mature companies [citation:6]

PEGY = (Price / EPS) / (Growth Rate + Dividend Yield)

⚡ Sharpe Ratio

Risk-adjusted return measure

Sharpe = (Return - Risk-Free) / Std Dev

📚 Price to Book

Compares market value to book value

P/B = Price / Book Value per Share

🔄 Return on Equity

Profitability relative to shareholder equity

ROE = Net Income / Shareholder's Equity

⚖️ Debt to Equity

Leverage ratio measuring financial risk

D/E = Total Liabilities / Shareholder's Equity
